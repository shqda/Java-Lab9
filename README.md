## Основное задание

Написать программу, приводящую к ситуации **взаимной блокировки (deadlock)**.

### Требования

* Использовать несколько потоков
* Создать ситуацию, при которой потоки:

  * захватывают ресурсы в разном порядке
  * и не могут продолжить выполнение

---

## Дополнительное задание

### Класс `MyRandomList`

Реализовать класс на основе `LinkedList` со следующими методами:

* `void addNumber()` — добавление случайного элемента
* `void removeNumber()` — удаление элемента со случайным индексом
* `int calcZero()` — подсчёт количества нулевых элементов

---

### Потоки

#### 🔹 `MyWriteThread`

* Добавляет или удаляет случайные элементы из `MyRandomList`

#### 🔹 `MyReadThread`

* Подсчитывает количество нулевых элементов
* Выводит результат на экран

---

### Синхронизация

Реализовать две версии `MyRandomList`:

* Без синхронизации
* С синхронизацией (использовать только `synchronized`)

---

### Требования

* В дополнительных заданиях использовать **только `synchronized`** (без других примитивов)
* В `main`:

  * запустить несколько потоков
  * подобрать их количество так, чтобы:

    * в версии **без синхронизации** возникала ошибка
    * в версии **с синхронизацией** всё работало корректно
